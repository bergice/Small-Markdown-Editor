# Small Markdown Editor

This small editor makes it easy to write and read markdown documents without any hassle. It is written in NW.js.

![img](../small-markdown-editor.png)

## Features
- Easy to use
- Compact
- Cross-platform
- No installer
- Responsive


## Library Dependencies
* color-scheme
* marked
